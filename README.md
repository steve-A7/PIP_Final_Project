# --Bangalore Dining Dynamics: Analyzing Ratings, Cost, and Popularity--

## I. Clone the repository:

```bash
https://github.com/steve-A7/PIP_Final_Project
```

## II. Extract Zip and Open Powershell/Command Prompt for the project directory:

```bash
cd Your:\\Save\\Path\\PIP_Mid_Project-main
```

## III. Create venv environment:

```bash
py -m venv .venv
```

##### or

```bash
python3 -m venv .venv
```

## IV. Activate venv environment:

### Windows

```bash
.venv\Scripts\activate
```

### MacOS/Linux

```bash
source .venv/bin/activate
```

## V. Install dependencies:

```bash
python -m pip install -r requirements.txt
```
